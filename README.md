# 记忆色块 (Memory Color Blocks)

一款面向儿童的记忆力训练 PWA 游戏。

## 功能特点

- 🎮 **难度选择模式**: 初级(2×2)、中级(3×3)、高级(4×4)
- 🏆 **挑战模式**: 无尽闯关，难度递增
- 🎴 **动物图卡收集**: 根据表现解锁各种动物图卡
- 📱 **PWA 支持**: 可安装到主屏幕，支持离线游玩
- 🎨 **精美界面**: 现代化 UI 设计，动画流畅

## 技术栈

- React 18
- Vite 6
- PWA (vite-plugin-pwa)
- 纯 CSS (无框架依赖)

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

## Docker 部署

```bash
# 使用 docker-compose 部署
docker-compose up -d

# 或直接构建运行
docker build -t mcb .
docker run -d -p 80:80 mcb
```

## 游戏规则

1. **记忆阶段**: 在倒计时内记住所有色块的位置
2. **游戏阶段**: 将打乱的色块拖拽到正确的格子中
3. **完成**: 填满所有格子后点击确认，查看成绩

## 动物图卡

根据难度、正确率和用时获得不同稀有度的动物图卡：

| 稀有度 | 颜色 | 示例 |
|--------|------|------|
| 普通 | 灰色 | 小蜗牛、小乌龟 |
| 稀有 | 蓝色 | 小兔子、机智狐狸 |
| 史诗 | 紫色 | 勤劳小蜜蜂、智慧猫头鹰 |
| 传说 | 金色 | 记忆大师象、闪电豹 |
| 神话 | 红色 | 神龙、独角兽 |
